## Getting Started
### Requirements
 1. [.NET Framework 6.x]
 2. [Visual Studio](https://visualstudio.microsoft.com/downloads/)

### Steps to Follow
 1. Open the .sln file with Visual Studio
 2. Build solution (Ctrl + Shift + B)
 3. Run the server (Ctrl + F5)
